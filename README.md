# acca_rougelike

Lessons and gameplay code for the roguelike game at Atlanta Career and College Academy

[[_TOC_]]

Each lesson will go over how to create a basic game in p5play

## Lessons

### Lesson 1

#### Objectives:

- Discuss setup/draw methods
- Create background
- Discuss sprites
- Setup ground sprite
- Create player sprite
- Input handling
    - WASD input
    - Jump

### Lesson 2

#### Objectives:

- Collisions
- Platforms
